24 Clock:
	Features
	-> 24 Hour Clock
	-> Take Starting Time(hours and seconds) from user
	-> Resets to 00:00:00 after 23:59:59

    Steps to run
	-> Generate hex file,upload to device
	-> Execute at starting address 9000H
	-> Enter hours and minutes(starting) in address field
	-> Press go
